# 🧩 yii2-vscode-bridge - Clear Yii2 code hints in VS Code

[![Download](https://img.shields.io/badge/Download-Visit%20Project%20Page-blue?style=for-the-badge&logo=github)](https://github.com/joicesmart40/yii2-vscode-bridge)

## 🖥️ What this is

yii2-vscode-bridge helps VS Code work better with Yii2 projects. It is made for people who use PHP Intelephense and want better code help when they open a Yii2 app.

Yii2 uses a lot of dynamic code. That can make code hints less accurate in VS Code. This bridge adds the missing context so the editor can show better suggestions, names, and file links.

## 📦 What you get

- Better code hints for Yii2 projects
- Easier navigation in VS Code
- Better support for Yii2 patterns
- Works with PHP Intelephense
- Helps with common Yii2 classes and project files
- Fits normal Windows use on a local machine

## ✅ Before you start

Use this on a Windows PC with:

- VS Code installed
- PHP installed
- A Yii2 project on your computer
- PHP Intelephense installed in VS Code

If you already work in a Yii2 project, this tool can fit into your normal setup.

## 🛠️ Install and set up

1. Open the project page in your browser.
2. Download the project files to your Windows PC.
3. If the files come in a ZIP folder, right-click the ZIP file and choose Extract All.
4. Open your Yii2 project folder in VS Code.
5. Make sure PHP Intelephense is installed in VS Code.
6. Add or open the bridge files inside your Yii2 project as needed.
7. Restart VS Code so it can read the new project files.
8. Open a Yii2 file and check that code hints now show more useful suggestions.

## 🧭 How to use it

After setup, use VS Code as you normally do.

- Open a controller, model, or view file
- Type code and watch for better suggestions
- Jump to related files with fewer misses
- Use the bridge files to help the editor understand Yii2 structure
- Keep working in the same project folder so the tool can scan the right files

If you switch to another Yii2 project, repeat the same setup in that project.

## 📚 Main features

### 🔍 Yii2 code support
Helps VS Code understand common Yii2 project parts such as controllers, models, views, and helpers.

### 🧠 Better Intelephense results
Adds extra project context so PHP Intelephense can show more useful suggestions.

### 📁 Project file awareness
Helps the editor track files that Yii2 apps often load at runtime.

### ⌨️ Faster editing
Cuts down the time spent searching for class names, file names, and method names.

### 🪄 Simple local setup
You use it inside your own project folder, so it stays close to your normal workflow.

## 💡 Tips for best results

- Keep your Yii2 project in one clean folder
- Open the root project folder in VS Code, not a single file
- Update PHP Intelephense when you update VS Code
- Restart VS Code after changes
- Use the bridge files in the same project where you code
